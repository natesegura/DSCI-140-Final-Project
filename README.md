## The Suicide Gender Disparity
Purpose: This project explores the gender disparity within global suicide rates. On average, male suicide rates are around 3 times as high as female suicide rates. Using basic data wrangling and visualization, this project aimed to increase understanding of why this disparity exists and locate patterns within OWID datasets.
